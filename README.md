# 🧠 MenuATS - Exercícios de Lógica em JavaScript

Este repositório reúne uma coleção de atividades práticas de lógica de programação desenvolvidas com JavaScript, com foco no treinamento de raciocínio lógico e resolução de problemas.

## 📌 Sobre o projeto

O **MenuATS** foi criado como um espaço para centralizar exercícios fundamentais de lógica, simulando situações comuns do dia a dia e operações matemáticas básicas.

Os exercícios são executados via terminal, utilizando interação com o usuário através da biblioteca **Clack/prompts**.

## 🛠️ Tecnologias utilizadas

- JavaScript (Node.js)
- Clack/prompts
- Git & GitHub

## ⚙️ Funcionalidades implementadas

O projeto contém diferentes exercícios, incluindo:

- ➕ Operações de calculadora (adição, subtração, multiplicação e divisão)
- 📐 Cálculo da área de um quadrado
- 📊 Cálculo de média de valores
- ⏳ Cálculo do tempo de vida em meses com base na idade
- 🌡️ Conversão de temperatura (Celsius para Fahrenheit)
